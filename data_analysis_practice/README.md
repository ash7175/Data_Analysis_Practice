# Data Analysis Practice

Practicing data analysis using pandas.

## Dataset
This project uses the Data Jobs dataset by Luke Barousse.
Download it from: https://huggingface.co/datasets/lukebarousse/data_jobs or load it directly in Python:

from datasets import load_dataset
dataset = load_dataset("lukebarousse/data_jobs")

Place the CSV file inside the `data/` folder before running the notebook.

## How to run
1. Install requirements: pip install -r requirements.txt
2. Launch Jupyter: jupyter notebook
3. Open the notebook in the notebooks/ folder
